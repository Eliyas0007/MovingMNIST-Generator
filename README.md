# MovingMNIST-Generator
In this repo, we are using MNIST dataset to generate MovingMNIST dataset.

Currently, only vertical movement is supported.

More features are coming soon!

## Tested Environment

- Python 3.7.11
- opencv 4.5.5.62
- torch 1.10.1
- torchvision 0.11.2

## How to use

```
$ python main.py
```
